Limpieza y transformacion de base de datos de netflix con python utilizando las librerias, pandas, matplotliby numpy. 
En el dataset de titlos de netflix se eliminaron datos que no contenian ningun valor, se elimino la columna que guardaba el ano en el que se agrego dicha pelicula a la plataforma, se cambio el nombre de la columna listed_in por category dejando asi solo el primer valor.
